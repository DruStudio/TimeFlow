# Time Flow ⏳

**A personal mindfulness tool for visualizing the passage of time**

## About

Time Flow is a beautiful web application that helps you visualize how much of the current year has passed through 10 unique, meditative visualizations.

## Features

- 10 different time visualizations
- Real-time calculations based on current date
- Light and dark mode
- Fully responsive design
- Share functionality
- Fullscreen viewing mode

## Technology

- Pure HTML, CSS, and JavaScript
- No frameworks or dependencies
- Optimized for performance
